### **What is Intellisort? How does it work?**
Intellisort is an AI-powered platform that analyzes your waste and provides efficient sorting and disposal solutions. Just take a picture of waste, and our AI will generate personalized disposal instructions for you!

### **Why should I use Intellisort?**
Using Intellisort makes waste management effortless and eco-friendly. It helps you reduce waste, recycle efficiently, and make sustainable choices.

### **Is Intellisort easy to use?**
Absolutely! Intellisort is designed to be user-friendly. Just snap a photo of waste you have, and get instant, easy-to-follow sorting and disposal suggestions.

Yes, Intellisort was developed as part of a high school initiative by students of The Millennium School, Amritsar. We're proud of our student team for creating such a useful tool!

By using Intellisort, you ensure that waste is disposed of correctly, reducing landfill use and promoting recycling. This helps conserve resources and lower carbon footprints.

## ğŸƒ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Gurkirat-Khaira/Intellisort.git

# Navigate to project directory
cd Intellisort

# Install dependencies
npm install
# or
yarn install
```

### Development Server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

### Build for Production

```bash
npm run build
npm start
```
